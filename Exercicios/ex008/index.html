Principal formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>. (não semantica)

Já nesta frase, temos um termo destacado usando a tag <strong>. (semantica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>. (não semantica)

Nesta frasem temos um termo em itálico usando a tag <em>. (semantica)

Texto marcado

Pemos criar também um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande, e um texto pequeno

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado

Tabém podemos sublinhado com a tag <u>. (não semantico)

Texto suprescrito

Para adicionar um texto suprescrito ex: x20-5 + 2, usa-se a tag <sup>

Texto sobrescrito

Para adicionar um texto sobrescrito ex:H 2O , usa a tag <sob>
