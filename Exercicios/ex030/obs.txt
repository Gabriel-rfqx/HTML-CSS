    propriedades de items do grid


    Para posicionar um item em uma grade de grid: 
    para as linhas/rows:

        grid-row-start
        grid-row-end

    para as colunas/columns:

        grid-colum-start
        grid-colum-end
    
    span (quanto um elemento vai expandir) ex:
    
        -- sem espan -- 
            grid-row-start: 2;
            grid-row-end: 4;
            grid-column-start: 3;
            grid-column-end: 5;

        -- com span --
            grid-row-start: 2;
            grid-row-end: span 3;
            grid-column-start: 3;
            grid-column-end: span 4;

    ShortHand

    grid-row: grid-row-start / grid-row-end
    grid-column: grid-column-start / grid-column-end

    grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end
